# cpp_examples - Cpp Example code and Unit Test using Google Test

## Prerequisite

* Java SDK Development
* Apache ant
* Google Test

## Directory structure

```sh

cpp_example
  |
  + src ( cpp files and Makefile )
  |
  + test
    |
    + src ( Unit Test files and Makefile )
    
```

## How to run it

```sh

cd app_examples

ant 

ant run

ant clean

ant run-test

```
